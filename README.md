# India Electric Vehicle Sales Analysis 

## Project Overview
This project analyzes electric vehicle sales trends across Indian states
using Python, Pandas, and data visualization techniques.

## Objectives
- Analyze EV sales trends in India
- Identify top states adopting EVs
- Study vehicle category distribution
- Visualize EV adoption patterns

## Dataset
Source: Government EV registration data / Vahan dashboard

## Tools & Technologies
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook

## Key Insights
- EV adoption is increasing every year
- Certain states dominate EV registrations
- Two-wheelers contribute the largest share of EV sales

## Visualizations
- EV Sales Trend Over Years
- State-wise EV Sales
- Category-wise EV Sales
- Heatmap Analysis

## How to Run
1. Clone repository
2. Install requirements
3. Run Jupyter Notebook
